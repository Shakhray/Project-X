Запуск программы. 
Основное окно.
Информация:
  Совершить операцию для начала работы с БД
Описание типов информации:
    Тип_1: нужна пользователю для принятия решения (подсказки диалоговых окон)
    Тип_2: нужна для осуществления операции (информация, вводимая  пользователем)
1. Доступные операции:
  - Выбрать существующую БД
  - Создать новую БД
   
2. Информация для осуществления операций
  Выбрать существующую БД:
      Тип_1: вывести информацию о существующих БД
        а) название
        б) описание
      Тип_2: (не требуется)
  Создать новую БД:
      Тип_1: (не требуется)
      Тип_2: запросить название БД, выбор места хранения, описание
      
После выбора БД
Окно состояния БД
  Информация:
    1. Список таблиц
    2. Представление таблицы
      - название
      - описание
    3. Структура таблицы
      - названия столбцов
      - типы столбцов
      - PK
    4. Представление столбца
      - название 
      - явзяется ли данный столбец PK
      - тип:
        a) наименование
        б) максимальное количество символов
      - описание
      - значение по умолчанию (?)
    4. Данные
      - наименование столбцов 
      - информация в столбцах построчно 
    5. Область конфликтов.
  Операции над:
    1. окном состояния БД
      - Сохранить все изменения
      - Отменить все изменения
    2. списком таблиц
      - Создать таблицу
      - Удалить таблицу
    3. структурой таблицы
      - сохранить изменения
      - отменить изменения
      - добавить столбец в структуру
    4. представлением столбца
      - сохранить изменения
      - отменить изменения
    5. данными
      - отменить все операции (? либо очистить все данные)

Область конфликтов расппологается внизу узким блоком во всю ширину экрана


Описание кнопок и элементов таблиц блоков:
1. Блок Список таблиц
	1. Создать - создаёт новую таблицу. По нажатию на кнопку Создать появляются блоки Структура, Представление столбца, Данные. Поля блоков пусты. Поля Название и Описание блока Состояние также БД пусты.
	2. Удалить - удаляет выбранный пункт из списка таблиц
        3. пункт списка Список таблиц - При нажатии на пункт из списка таблиц появляются блоки Структура, Представление столбца, Данные. Поля блоков заполняются информацией.
2. Блок Структура
	1. Сохранить - сохраняет введенное значение
	2. Отмена - очищает поля ввода при добавлении данных, откатывает данные назад при их редактировании.
	3. (?) Добавить - добавляет новую запись в блок (подумать о её целесообразности)
	4. строка таблицы Структура - при нажатии на строку таблицы Структура обновляется блок Представление столбца
3. Представление столбца
	1. Сохраннить - сохраняет введенные значения
	2. Отмена - очищает поля ввода при добавлении данных, откатывает данные назад при их редактировании.
4. Данные
	1. Отмена (не знаю, что она будет отменять. Можно подумать над кнопкой Очистить всё)
	2. Ячейка таблицы. Становится доступной при установки на неё фокуса. При снятии фокуса прекращается её редактирование, введенные данные сохраняются.
5. Состояние БД
	1. Сохранить - создает новую таблицу в БД, если таблица создавалась вновь, и сохраняет изменения в существующую таблицу при её редактировании. 
	   В таблицу записываются все данные из блоков Структура, Представление столбца, Данные, Состояние БД.
