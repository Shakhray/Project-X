Структура таблиц для хранения информации о схеме БД (т. е. структуре целевых таблиц).
Эти таблицы организуются в БД, создаваемой в Редакторе БД.

1. table_info    Основная информация о таблице.

   1. id
   2. name (table_name)   
   3. description
   
2. column_info

   1. id                       
   2. name (column_name)
   3. description
   4. data_type               ссылка на тип данных (таблица data_type)
   5. data_type_par1          первый параметр типа
   6. data_type_par2          второй параметр типа
   7. in_pk                   признак принадлежности первичному ключу
   8. def_value_origin        значение по умолчанию при создании записи        ???
   9. def_value_transform     значение по умолчанию при преобразовании типов
   
   10. table_id               ссылка на table_info - таблицу, которой принадлежит столбец
   
3. data_type

   1. id
   2. name
   3. parameter_count         количество параметров типа
   ...  Добаваить данные, необходимые при обработке типов в процессе работы программы.
