Редактор баз данных. Версия 1.

===================================================
Функции
===================================================
1. Построение структуры БД.
   Ограничиваемся таблицами, т. е. никаких представлений, хранимых процедур и т. д.

   Должен быть список таблиц БД.
   Операции на нём:
   1. Добавление повой таблицы.
   2. Изменение существующей.
   3. Удаление таблицы.

   При изменении и вводе новой таблицы рассматриваем минимальный набор данных:
   1. Название таблицы.
   2. Набор столбцов:
      1. Название столбца.
      2. Тип данных столбца.
      3. Индекс.
         Можно пока ограничиться первичным ключом.
      4. Описание столбца.
   3. Описание таблицы.

2. Работа с сданными.
   В первой версии предусамтриваем только самые пеобходимые операции с данными.
   Во-первых, операции производятся над данными одной таблицы.
   Во-вторых, реализуем пока только операции над отдельными строчками таблицы.

   Должен быть список таблиц.
   Операции:
   1. Выбор таблицы для просмотра данных.

   При выборе таблицы:
   1. Добавление строчки.
   2. Удаление строчки.
   3. Изменение строчки.

===================================================
Дополнительные требования
===================================================
1. Возможность работы с несколькими оболочками БД. (Оболочка БД - смотри "Термины")
   
2. Инфраструктура для расширения данных о таблице.
   В дальнейшем предполагается привязывать к таблицам (и её столбцам) дополнительную информацию:
   интерпретации, связи с другими таблицами и т. д.

Замечание: программа разрабатывается для собственного пользования, поэтому необходима максимальная
           автоматизация процесса работы с БД.
